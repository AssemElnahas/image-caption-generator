## Image Caption generation model not pretrained
This is basically a collection of prompts for making good image captions using AI models that handle pictures. They have been tested a lot and work well for different needs.

I like how it points out that regular captions from these models can come out bland or full of keywords in a weird way. The prompts here fix that by making things sound more natural, like how a person would describe it. For example, some are poetic for blogs or social posts, others are straight facts for websites or shopping sites. There is one style for short punchy ones that could go viral on Instagram or Twitter. And detailed ones for analyzing art or photos in research.

It works with models like GPT-4o or Claude, even open ones like LLaVA. I am not sure if it covers every model out there, but the ones mentioned seem solid.

Inside, there are files in a prompts folder. Basic one is good to start with for normal stuff. Detailed is for describing photos like in movies or stories, maybe product images too. Short viral keeps it under 15 words and engaging. Accessible is neutral and factual, good for alt text so people with disabilities can use it, or for catalogs. No meta avoids saying things like photo of or image shows, which cleans it up. Then examples folder has about 30 real images with prompts and outputs, some good some not, to see what happens. And a big file with templates you can tweak.

To get going quick, just copy a prompt, paste into the model, add your image. Sometimes add extra at the end if you want, but it is optional. That part gets a bit messy, like not everything is spelled out fully.
